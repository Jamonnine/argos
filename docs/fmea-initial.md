# ARGOS FMEA (Failure Mode & Effects Analysis) 기초 문서

> v1.0 — 2026-03-16 | 대상: ARGOS v2.0 8중 센싱 체계
> 목표 SIL 등급: **SIL 2** (IEC 61508 기준)

---

## 1. 시스템 경계

```
ARGOS 시스템 = 오케스트레이터 + 8중 센싱 + Nav2/SLAM + 드론/UGV 플랫폼
외부 인터페이스: 웹 대시보드 (rosbridge), 소방 지휘관 (수동 오버라이드)
```

## 7. ICS/NIMS 참조

- **미국 NIMS**: [FEMA NIMS Doctrine](https://www.fema.gov/emergency-managers/nims)
- **한국 소방청 ICS**: 소방청 현장지휘체계 운영 지침 (소방청 훈령)
- **ARGOS 매핑**: 오케스트레이터(지휘관) → 팀장(로봇 리더) → 대원(개별 로봇)
- **소방청 10대 전략과제** (2026.01.16): 5번(이종 무인장비) + 4번(AI 현장지휘) = ARGOS
