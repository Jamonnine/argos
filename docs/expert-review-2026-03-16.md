## 전 전문가 공통 지적 TOP 5

### 1. ~~CI/CD 부재~~ → ✅ GitHub Actions + Dockerfile 생성 (DevOps 34→58)

### 2. ~~센서 퓨전 가중 합산 없음~~ → ✅ compute_situation_score() 구현 (퓨전 60→70)
- 5개 센서 가중치: 가스(0.30) > 화재(0.25) > 구조물(0.20) > 피해자(0.15) > 음향(0.10)

### 3. 내열/방수/하드웨어 현실성 (HW 58) — **미개선 (설계 단계 필요)**
- 내열 설계 2.1/10 (전 평가 최저 개별 점수)
- 배터리: 드론에만 RTL 모델 추가, UGV는 미개선

### 4. ~~지휘관 승인 체계 부재~~ → ✅ PAUSED→승인→RETURNING 구현 (소방 83→89)
- gas critical → 자동 RETURNING 대신 PAUSED + evacuation_recommended
- resume 서비스로 지휘관 승인 시 RETURNING 전환

### 5. 현장 통신 환경 대응 부족 (통신 63) — **미개선 (Zenoh 장기)**
