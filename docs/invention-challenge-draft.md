> **작성일**: 2026-03-23
> **발명명**: ARGOS — 이종 군집 소방 로봇 자율 오케스트레이션 시스템
> **발명자**: 민발 (대구 강북소방서 소방교)
> **분류**: 소방장비/ICT 융합

## 1. 발명 개요

### 해결하려는 문제
현재 소방 로봇(HR-셰르파 등)은 **원격 조종 단일 운용**으로, 화재 현장의 복잡성에 비해 활용도가 제한적입니다.
- 1대 로봇 = 1명 조종사 필요 → 다수 로봇 동시 운용 불가
- 통신 두절 시 로봇 완전 정지 → 현장 방치 위험
- 드론과 지상 로봇의 정보 공유 체계 부재

### 해결 방안
**ARGOS**는 드론, UGV(차량형), HR-셰르파(소방로봇)가 **한 팀으로 자율 협업**하는 오케스트레이션 플랫폼입니다.
- 지휘관이 "무엇을" 지시 → 로봇이 "어떻게"를 자율 수행 (Supervised Autonomy)
- CBBA 경매 알고리즘으로 **최적 임무 자동 배정**
- 통신 두절 시 **로컬 자율 판단** 모드 자동 전환

## 2. 핵심 기술

### 2.1 이종 군집 오케스트레이션
```
현장 지휘관 (인간)
    ↓ "건물 2층 탐색해"
ARGOS 오케스트레이터
    ↓ CBBA 경매 할당
┌────────┬────────┬────────┐
│ UGV #1 │ UGV #2 │ 드론   │
│ 1층 탐색│ 2층 진입│ 상공감시│
└────────┴────────┴────────┘
```

- **4계층 아키텍처**: Orchestrator → Mission Modules → Core Services → Platform
- **PlatformInterface 추상화**: UGV/드론/셰르파를 동일 인터페이스로 제어
- **CBBA 태스크 할당**: 로봇의 능력(열화상, 비행, 방수포)을 고려한 경매 기반 최적 배정

### 2.2 HR-셰르파 호스 제약 모델링 (세계 최초)
- **100m 호스 릴** 물리 모델: 잔여 길이, 꺾임 위험도, 충수 후진 금지
- **Hose-Aware Path Planning**: 급커브 감지 + 호스 경로 장애물 등록
- **멀티 셰르파 호스 간섭 감지**: 2대 호스 교차 시 자동 역할 재분리

### 2.3 드론→UGV 화재 핸드오프
- 드론이 열화상으로 화점 감지 → 자동으로 UGV 진압팀 배정
- 드론은 상공 감시 유지, UGV는 지상 진입 진압
- **쿨다운 + 근접 중복 제거**로 오탐 방지

### 2.4 AI 화재 탐지
- YOLOv8 기반 화재/연기 탐지 (mAP50=0.715)
- NVIDIA Omniverse 합성 데이터 2,030장 + FLAME Diffuser 1,000장 학습
- AGPL 라이선스 분리 (별도 패키지 argos_fire_ai)

## 3. 차별성

| 구분 | 기존 | ARGOS |
|------|------|-------|
| 로봇 운용 | 원격 조종 (1:1) | 자율 군집 (1:N) |
| 로봇 종류 | 단일 | 이종 (UGV+드론+셰르파) |
| 임무 배정 | 수동 | CBBA 경매 자동 |
| 통신 두절 | 로봇 정지 | 로컬 자율 판단 |
| 호스 관리 | 수동 | 물리 모델 자동 제약 |
| 소방 지휘체계 | 미매핑 | 소방 전술 직접 매핑 |

### 전 세계 유일
- **이종 군집 소방 로봇** 상업 배치 0건 (블루오션)
- **호스 제약 반영 경로 계획** 학술 논문 0건
- **소방관이 직접 설계한** 소방 지휘체계 매핑
