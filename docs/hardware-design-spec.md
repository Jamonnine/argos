## 1. 내열/방수 설계 (IP54 목표)

### 1.1 내열 설계

| 구성요소 | 작동 온도 | 보호 방법 | 참조 |
|---------|---------|---------|------|
| LiDAR (RPLIDAR A2) | 0~50°C | 세라믹 섬유 차열판 + 공기 gap | HR-셰르파: 800°C 내열 |
| RGB 카메라 | 0~35°C | 사파이어 윈도우 + 냉각팬 | — |
| 열화상 | 0~70°C | 자체 내열 (적외선 센서) | — |
| 배터리 (LiPo) | -20~60°C | 단열재 감싸기 + 온도 센서 | — |
| 전자보드 | 0~70°C | 알루미늄 방열판 + 밀폐 박스 | — |

### 1.2 방수 설계 (IP54)

| 등급 | 의미 | 적용 |
|------|------|------|
| IP5X | 분진 방호 (유해한 양 침입 방지) | 밀폐 전자 박스 + 방진망 |
| IPX4 | 모든 방향 물 튐 방호 | 실리콘 실란트 + 방수 커넥터 |

### 1.3 화학물질 저항

- 렌즈 보호 윈도우: 광학 사파이어 (HCl 내성)
- 몸체: 알루미늄 6061 (양극산화 처리)
- 센서 에어 퍼지: 압축 공기로 분진 제거 (선택)

## 3. 질량 재검증

### 3.1 현재 URDF vs 실제 예상

| 부품 | URDF (현재) | 실제 예상 | 차이 |
|------|-----------|---------|------|
| 섀시 | 15.0 kg | 8.0 kg | -7.0 kg |
| 바퀴 × 4 | 4.0 kg | 2.0 kg | -2.0 kg |
| 센서 (5종) | 0.7 kg | 1.2 kg | +0.5 kg |
| 배터리 | 0 kg | 0.8 kg | +0.8 kg |
| 전자부품 | 0 kg | 1.5 kg | +1.5 kg |
| 차열판/방수 | 0 kg | 1.0 kg | +1.0 kg |
| **총합** | **19.7 kg** | **14.5 kg** | -5.2 kg |

### 3.2 권장 URDF 수정

```xml
<!-- dimensions.xacro 수정 권장 -->
<xacro:property name="base_mass" value="8.0"/>     <!-- 15.0 → 8.0 -->
<xacro:property name="wheel_mass" value="0.5"/>    <!-- 1.0 → 0.5 -->
<!-- 추가: 배터리 링크 0.8kg, 전자부품 링크 1.5kg -->
```

## 5. Legged 플랫폼 인터페이스 설계

### 5.1 요구사항

| 항목 | UGV (현재) | Legged (확장) |
|------|-----------|-------------|
| 이동 방식 | 4WD 차동 구동 | 4다리 × 2~3 DOF |
| cmd_vel | linear.x + angular.z | 동일 (보행 컨트롤러가 변환) |
| TF | base_footprint → base_link | 동일 (다리는 내부 TF) |
| 센서 마운트 | 센서 플랫폼 (고정) | 동일 (몸체 상단) |

### 5.2 인터페이스 호환성

```
오케스트레이터
    ↓ cmd_vel (Twist)
[Platform 추상화 레이어]
    ├── UGV: diff_drive_controller
    ├── Drone: MulticopterVelocityControl / PX4 offboard
    └── Legged: walking_controller (ROS 2 ros2_control 기반)
```

**핵심**: `cmd_vel` 토픽은 동일. 플랫폼별 컨트롤러가 다리/바퀴/로터로 변환.
오케스트레이터는 플랫폼 타입을 모르고, `capabilities`만 질의.

## 6. 드론 열화상 카메라 SDF 추가 (참조)

```xml
<!-- model.sdf에 추가 -->
<sensor name="thermal_camera" type="thermal_camera">
  <pose>0 0 -0.08 0 1.5708 0</pose>  <!-- RGB 아래 8cm -->
  <always_on>true</always_on>
  <update_rate>15</update_rate>
  <camera>
    <horizontal_fov>1.5708</horizontal_fov>  <!-- 90° 광각 -->
    <image>
      <width>256</width>
      <height>192</height>
      <format>L8</format>
    </image>
    <clip><near>0.5</near><far>100</far></clip>
  </camera>
  <topic>thermal_down/image_raw</topic>
</sensor>
```
