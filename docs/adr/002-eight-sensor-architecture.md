# ADR-002: 8중 센싱 아키텍처 채택

- **상태**: 승인
- **일시**: 2026-03-16
- **결정자**: 민발

## 맥락
단일 센서(열화상만)로는 화재 현장의 복합 위험을 감지할 수 없다. 화재 사망의 70~80%는 화염이 아닌 CO 중독과 질식.

## 결정
8중 센싱 체계를 구현한다:
1. 열화상 (화점 위치)
2. 불꽃/연기 감지 (YOLO AI)
3. 화재현장 물체 감지 (AI-Hub 15종)
4. 연기 감지 (SlowFast 영상 분류)
5. 가스 (CO/O2/LEL/CO2/HCN)
6. 피해자 (열화상 인체)
7. 구조물 안전성 (LiDAR 변위)
8. 음향 (폭발/가스누출/구조요청/붕괴)

## 근거
- 소방 현장 다중 위험 동시 존재 (화재+가스+구조물 붕괴+피해자)
- 단일 센서 오탐/미탐을 다중 센서 교차검증으로 방어
- NIOSH/OSHA 가스 기준, 소방청 10대 전략과제 정합

## 결과
- 오케스트레이터가 8개 센서 콜백으로 종합 판단
- 센서 퓨전 가중 합산 (compute_situation_score)
- temporal smoothing (2프레임 연속 확인)
- 우선순위 행렬 (gas_critical > structural_critical > explosion > victim > fire)
