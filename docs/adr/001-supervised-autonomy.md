# ADR-001: Supervised Autonomy 아키텍처 채택

- **상태**: 승인
- **일시**: 2026-03-04
- **결정자**: 민발 (프로젝트 리드)

## 맥락
소방 로봇은 완전 자율과 완전 원격 조종 사이에서 균형이 필요하다. 완전 자율은 예측 불가능한 소방 현장에서 위험하고, 완전 원격 조종은 통신 두절 시 무력하다.

## 결정
**Supervised Autonomy** (감독 자율성) 패턴을 채택한다.
- 오케스트레이터가 "무엇을" 결정 (임무 할당, 단계 전환)
- 각 로봇이 "어떻게"를 자율 수행 (경로 계획, 장애물 회피)
- 통신 두절 시 마지막 임무를 독립 수행

## 근거
- DARPA SubT Challenge 우승팀 (CERBERUS, CoSTAR) 동일 패턴 사용
- 소방 지휘체계(ICS)의 "지휘관→팀장→대원" 구조와 자연스럽게 매핑
- 통신 불안정 환경에서도 기본 동작 보장

## 결과
- 오케스트레이터 노드가 중앙 집중식 지휘
- 각 로봇은 Nav2 기반 자율 내비게이션
- 2026-03-16: 지휘관 승인 체계 추가 (gas critical → PAUSED → 승인 → RETURNING)
