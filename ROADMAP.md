# ARGOS Roadmap

> **최종 업데이트**: 2026-03-15
> **현재 버전**: v1.0 (MS-1~10 완료) → v2.0 개발 진행 중
> **누적 개발**: ~110시간 / 239 테스트 / 7 노드 / 10 launch / 10 커스텀 인터페이스

---

## 프로젝트 비전

**이종 군집 소방 로봇 오케스트레이션 플랫폼** — 드론, UGV, 보행형 로봇이 한 팀으로 화재 현장을 자율 탐색하고, 화점을 감지하며, 실시간으로 정보를 공유하는 Supervised Autonomy 시스템.

### 핵심 차별점
- **이종(Heterogeneous)**: 단일 플랫폼이 아닌 드론+UGV+보행형 조합 — 각 플랫폼의 강점 결합
- **오케스트레이션**: "무엇을" 결정하는 지휘관 + "어떻게"를 자율 수행하는 로봇
- **현장 기반 설계**: 현직 소방관이 실제 소방 지휘체계를 소프트웨어로 매핑
- **오픈소스**: Apache 2.0 — 공공기관·수상·상업 사용 가능

## 정책 정합성 (2026-03 기준)

ARGOS가 구현한 기능들이 소방청 공식 정책 방향과 정확히 일치한다.

| 소방청 10대 전략과제 (2026.01.16) | ARGOS 대응 | 상태 |
|----------------------------------|-----------|------|
| **5번 — 이종 무인장비 조합** (육상로봇+드론+수상정 입체작전) | 4계층 아키텍처 Platform 추상화 (MS-8) | 구현 완료 |
| **4번 — AI 현장지휘체계** (드론·로봇 정보 → AI 분석 → 자원 배치) | 오케스트레이터 노드 (MS-7) | 구현 완료 |
| HR-셰르파 100대 보급 (2027~2030) | 100대 원격조종 불가 → 분산 오케스트레이터 필수 | ARGOS가 해결 |
| 드론 출동 연 40% 성장 (2024년 4,623건, 676대) | 다수 드론 오케스트레이션 수요 근거 | ARGOS가 해결 |
| 소방청 R&D 예산 503억 (전년 대비 +65%) | R&D 공모(IRIS) 참여 기회 | 모니터링 |

**출처**: 소방청 보도자료 (2026.01.16), 이데일리, 아시아경제, 아주경제

## 경쟁 환경 분석

| 구분 | 현황 | ARGOS 포지셔닝 |
|------|------|---------------|
| HR-셰르파 (현대로템) | 원격 조종, 단일 로봇, 대당 20억 | ARGOS = 자율 오케스트레이션 소프트웨어 (HR-셰르파 위에 올릴 수 있음) |
| 서울소방 AI 순찰 로봇 | 단일 로봇, 특정 시나리오 | ARGOS = 범용 군집 플랫폼 (순찰도 시나리오의 하나) |
| 대구 AI 군집드론 (ETRI) | 드론만, 산불 특화 | ARGOS = 이종(드론+UGV) + 건물 진입 가능 |
| 전 세계 스웜 로봇 | 상업 배치 사례 0건 | ARGOS = 반자율 현실적 접근 (Supervised Autonomy) |

**핵심**: 단일 로봇도, 동종 군집도 아닌 **이종 군집 오케스트레이션**이 ARGOS의 유일한 차별점. 이 방향이 소방청 10대 과제 5번과 정확히 일치하며, 전 세계적으로 아직 상업 배치 사례가 없는 블루오션이다.
